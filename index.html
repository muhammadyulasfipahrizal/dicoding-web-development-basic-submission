Sejarah HTML

HTML (Hypertext Markup Language) adalah bahasa markah standar yang digunakan untuk membuat dan mengatur konten pada halaman web. Sejarah HTML dimulai pada awal 1990-an ketika Tim Berners-Lee menciptakan World Wide Web di CERN (Organisasi Riset Nuklir Eropa).

Versi pertama HTML, disebut HTML 1.0, dirilis pada tahun 1993. Ini adalah versi awal HTML yang digunakan untuk membuat struktur dasar halaman web dengan elemen dasar seperti judul, paragraf, tautan, dan daftar.

Setahun kemudian, pada tahun 1994, HTML 2.0 diperkenalkan. Versi ini memperkenalkan fitur-fitur baru seperti formulir, tabel, dan gambar. HTML 2.0 juga menggantikan sejumlah elemen dan atribut dari HTML 1.0.

Pada tahun 1997, HTML mencapai versi 3.2. Versi ini memperkenalkan fitur-fitur baru seperti tabel yang lebih kompleks, daftar terurut dan tidak terurut, serta elemen-elemen yang lebih baik untuk mengatur tata letak halaman. HTML 3.2 juga diperluas melalui spesifikasi Non-Standard yang disebut "HTML +".

Kemudian, pada tahun 1999, HTML mencapai versi 4.01. Versi ini menghadirkan perubahan signifikan dalam hal aturan markup dan validasi dokumen. HTML 4.01 juga memperkenalkan fitur-fitur seperti CSS (Cascading Style Sheets) untuk mengatur tampilan halaman web.

Selanjutnya, dalam perkembangan HTML, muncul HTML5. HTML5 adalah versi terbaru dan paling canggih dari HTML. Pada tahun 2014, W3C (World Wide Web Consortium) merilis spesifikasi HTML5 yang menyertakan sejumlah fitur baru dan perbaikan dari versi sebelumnya. HTML5 menambahkan elemen-elemen baru seperti <video>, <audio>, <canvas>, <header>, <footer>, dan banyak lagi. Ini juga menyertakan fitur-fitur seperti dukungan untuk penyimpanan lokal, akses ke perangkat keras, dan pengembangan aplikasi web yang lebih interaktif.

HTML5 juga mendukung pemisahan struktur dan gaya dengan menggunakan CSS. Ini memungkinkan pengembang web untuk lebih fleksibel dalam mengatur tampilan dan tata letak halaman mereka.

Sejak rilis HTML5, spesifikasi HTML terus diperbarui oleh W3C dan diperluas dengan fitur-fitur baru. Versi-versi terbaru HTML masih menjadi standar dalam pengembangan web saat ini.

Sejarah CSS

Sejarah CSS dimulai setelah HTML mencapai versi 4.01 pada tahun 1999. Pada saat itu, HTML telah menjadi bahasa markah standar untuk membuat struktur dasar halaman web, tetapi masih terbatas dalam hal mengatur tampilan dan gaya halaman.

Dalam upaya untuk memisahkan antara struktur dan gaya halaman web, W3C (World Wide Web Consortium) mengembangkan CSS. CSS dirancang untuk memungkinkan pengembang web mengontrol tampilan elemen-elemen HTML dengan lebih mudah dan fleksibel. Dengan menggunakan CSS, pengembang dapat membuat perubahan gaya secara konsisten di seluruh situs web dengan mengubah beberapa aturan CSS, tanpa harus mengubah setiap elemen individu secara manual.

Versi pertama CSS, CSS1, dirilis oleh W3C pada tahun 1996. CSS1 menyediakan seperangkat aturan dan properti yang dapat digunakan untuk mengubah tampilan elemen HTML. Meskipun CSS1 masih terbatas dalam fitur dan kemampuannya, itu membuka jalan bagi perkembangan lebih lanjut dari CSS.

Seiring waktu, CSS mengalami pengembangan dan peningkatan. Versi-versi CSS yang lebih baru seperti CSS2, CSS2.1, dan CSS3 diperkenalkan dengan peningkatan fitur dan kemampuan yang signifikan. CSS3 memperkenalkan modul-modul baru seperti selektor lanjutan, animasi, transisi, dan fleksibilitas tata letak yang lebih baik.

Saat ini, CSS terus berkembang dengan munculnya CSS4 dan perkembangan terbaru dalam spesifikasi CSS. CSS telah menjadi bagian penting dari pengembangan web modern, memungkinkan pengembang untuk menciptakan tampilan dan gaya yang menarik, responsif, dan konsisten di seluruh berbagai perangkat dan browser.

Sejarah Javascript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat dan mengatur konten pada halaman web. Sejarah JavaScript dimulai pada awal 1990-an ketika Brendan Eich menciptakan bahasa tersebut di Netscape Communications Corporation.

Versi pertama JavaScript, disebut LiveScript, dirilis pada tahun 1995. Ini adalah versi awal JavaScript yang digunakan untuk memberikan interaktivitas pada halaman web dengan fitur dasar seperti manipulasi elemen HTML, validasi form, dan pengolahan data.

Setahun kemudian, pada tahun 1996, JavaScript diperkenalkan oleh Netscape sebagai bagian dari standar web yang disebut ECMAScript. Versi ini memperkenalkan fitur-fitur baru dan perbaikan dari LiveScript.

Pada tahun 1999, ECMAScript mencapai versi 3. Versi ini membawa perubahan signifikan dalam hal aturan dan fungsi bahasa JavaScript. ECMAScript 3 juga diperluas melalui spesifikasi Non-Standard yang disebut "JavaScript 1.5".

Kemudian, pada tahun 2009, ECMAScript mencapai versi 5. Versi ini menghadirkan perbaikan yang lebih lanjut dalam bahasa JavaScript dan memperkenalkan fitur-fitur baru seperti JSON (JavaScript Object Notation), metode Array baru, dan metode String baru.

Selanjutnya, dalam perkembangan JavaScript, muncul ECMAScript 6 (juga dikenal sebagai ECMAScript 2015). Versi ini dirilis pada tahun 2015 dan menyertakan banyak perubahan dan peningkatan signifikan dalam bahasa JavaScript, termasuk penambahan sintaks baru seperti arrow function, class, module, dan fitur-fitur lainnya.

ECMAScript 6 juga memperkenalkan fitur-fitur seperti let dan const untuk deklarasi variabel, template literals untuk string interpolation, dan destructuring assignment untuk memecah objek atau array menjadi variabel terpisah.

Sejak rilis ECMAScript 6, spesifikasi ECMAScript terus diperbarui dengan versi-versi baru seperti ECMAScript 7, ECMAScript 8, dan seterusnya. Setiap versi membawa perubahan dan peningkatan dalam bahasa JavaScript untuk mendukung pengembangan web yang lebih modern dan kompleks.
